The website requires the teacher to give the 4 digit number listed in the “Daily Password & Start Times” page to the students. The students will then enter their email and the 4 digit number into the “Attendance Submit” page to log their attendance.

Accounts

Firstly, teachers have to make an “account” in the website in order to access the pages. It requires an account to make an account, but you should be able to use the account details given to you.

Setting up the Emails and Start Times

The teacher has to set up the list of email addresses of their class. This can be done in the “Insert New Student” page and accomplishing the form.

Entries can also be removed from the database through the “Delete Student” page by typing in the database ID (DB ID) listed in the “Student Database” page.

Afterwards, the teacher also has to set the start time for their respective grade level. The start time indicates after what time a student is considered late. In the case of a start time of 7:00 a.m, any submissions after 7:00 will be considered as late.

After doing these steps, the set up for the website is done.

The Daily Password

The website generates a 4 digit number daily that the students have to submit alongside their email in order to log their attendance. Ideally, the teacher writes the 4 digit number on the whiteboard or in the classroom, so the students will be required to be in the classroom in order to be able to log their attendance.

Submitting Attendance

In the “Attendance Submit” page, there is a form that asks for your email and a daily password. The teacher will provide the daily password in the form of a 4 digit number.

Attendance Submit

This is just a form for the students to submit their attendance that will get logged into the database connected to this website.

Daily Password & Start Times

An account is required for this, so only teachers should be able to see it. This is where the website shows the 4 digit number for the day or the daily password. It is also where the teachers can set the time that their classes start in order to make the website accurate in logging lates.

Daily Attendance

This shows a table for who is late and who is absent today. It has a filter for grade and section to make going through the entries easier.

Student Database

Similar to the “Daily Attendance” page, this shows a table of students. It displays their info, such as their total number of lates, absences, and SPRs. It also has a filter for grade level and section.

Create/Delete Student

These pages are to add or delete entries of students in the database.

Account

These pages are to do things related to the teachers’ accounts in this website. It allows you to make a new account and to log in and out.
